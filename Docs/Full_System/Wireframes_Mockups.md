# Wireframes & Mockups Descriptions

The following outlines the core screen structures for the Web/App platform interfaces catering to the distinct workflows and actors defined in the BRD and SRS.

## 1. Web Portal: Admin Dashboard
**Purpose**: Central hub for system configuration, user management, and high-level KPIs.
- **Header**: Navigation menu (Dashboard, User & Teams, Channels, Compliance SOPs, Billing, Settings), Admin Profile.
- **Main Content (Dashboard Tab)**:
  - Top KPI Widgets: Total Revenue, RPM Average, Active Creators, Videos Published this week.
  - Middle Graph: Revenue & Views over time (filters by channel/team/date).
- **Users & Teams Tab**: List view of users with Roles, Status (Active/Deactivated), Team assignment dropdowns, and "Create User" action button.

## 2. Web Portal: Content Planning Board (Topic Selector)
**Purpose**: A kanban or calendar interface to manage AI topics.
- **Left Sidebar**: "AI Suggestions Inbox" - list of 10-20 daily topic cards generated by PlannerAgent. Each card displays Title, Category, Region Tag.
- **Main Area**: Weekly/Monthly Calendar View.
- **Actions**: Drag-and-drop a topic from the inbox to the calendar to "Approve" and schedule it. Click "X" on a card to "Reject" it.

## 3. Web Portal: AI Script Editor (Script Creator)
**Purpose**: A rich-text drafting environment paired with AI assistance.
- **Header**: Topic Title, Status (Draft, Finalized), "Send to Cameraman" button.
- **Left Panel**: AI Script settings (Tone, Length, Template selection) and "Re-Generate" button.
- **Main Panel**: The Editor. Sections clearly marked: `[HOOK]`, `[STORYLINE]`, `[SHOTLIST]`, `[DIALOGUE]`.
- **Right Panel**: Version history and suggestions.

## 4. Web Portal: Media Upload & Editor Workspace (Cameraman/Video Editor)
**Purpose**: Handling of large raw files and timeline prep.
- **Header**: Associated Script/Task info.
- **Upload Zone**: Drag-and-drop area showing progress bars and file validation messages (e.g., "File too large").
- **Asset Library**: Grid of uploaded clips. Hover options to apply Tags: "Good", "Retake", "Noisy".
- **AI Timeline Gen Panel**: Button to "Run EditorAgent". Displays resulting timeline JSON visual blocks.
- **Export/Upload Area**: Upload the final finalized video file.

## 5. Web Portal: QA & Publishing Review (Reviewer / SEO)
**Purpose**: Final check of metadata, video, and legal compliance.
- **Top Section**: Video Player with AI QC overlay markers (red marks on timeline for Faces, Logos).
- **Right Sidebar**:
  - **QC Report**: List of AI flags. Actions: `[Mark Fixed]`, `[Ignore]`.
  - **Metadata Form**: AI suggested Titles (radio buttons), Description textbox, Tags.
- **Bottom Actions**: `[Approve & Schedule]`, `[Reject + Reason Input]`.

## 6. Community Creator App (Mobile/Web)
**Purpose**: Simple, engaging portal for independent creators to make and monetize content.
- **Home Tab**:
  - Carousel of "AI Daily Topic Ideas".
  - Big CTA: "+ Create Video" (triggers AI Script -> Record -> AI Auto-edit flow).
- **My Content Tab**: List of uploaded videos, their QC status, and link to YouTube status. Notifications for "Faces detected - Auto-blur applied".
- **Earnings Tab**: Big bold number showing current unpaid balance. Graphs showing daily YouTube revenue. CTA: "Withdraw to Bank/Momo" (disabled if < $10).

## 7. Shop Owner (Camera Partner) Mobile App
**Purpose**: Easy management of physical camera permissions and content approval.
- **Dashboard**: Live feeds of Kitchen/Garden cameras. Alert: "New Edit Awaiting Approval".
- **Approval Queue Tab**: List of short videos cut from camera feeds. Swipe Right to Approve (gets published), Swipe Left to Reject.
- **Earnings Tab**: Revenue share earned from approved videos. Withdraw functionality.
- **Settings Tab**: Location verification, "Pause Camera" toggle, "Request Video Removal" form.

## 8. Freelancer Job Board
**Purpose**: Gig economy style interface for external tasks.
- **Discover Tab**: Feed of available tasks (e.g., "Edit a 10m video"). Tags for skill reqs.
- **Task Detail Sheet**: Script details, deadline, payout amount, `[Accept Job]` CTA.
- **My Jobs Task Workspace**: Where freelancers upload deliverables, view chat threads with Reviewers, and click `[Submit Invoice]`.
